### Discord Bot Premium Indonesia

Bot Discord multifungsi yang dibangun menggunakan **Discord.js v14** dan **MongoDB**, menghadirkan fitur moderasi, ekonomi, musik, AI, anime, hingga voice channel temporary dalam satu bot.

# ⚙️ Instalasi

## Clone Repository

```bash
git clone <repository-url>
cd Lunaria
```

---

## Install Package

```bash
npm install
```

---

## Konfigurasi .env

```env
# Discord
DISCORD_TOKEN=
CLIENT_ID=
GUILD_ID=

# Prefix
PREFIX=ln!

# Database
MONGO_URI=

# Embed
EMBED_COLOR=#2b2d31

# AI
AI_API_KEY=

# Spotify
SPOTIFY_CLIENT_ID=
SPOTIFY_CLIENT_SECRET=
```

---

## Jalankan Bot

```bash
npm start
```

atau

```bash
node index.js
```
